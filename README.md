# 📊 HR Recruitment Dashboard (Power BI)
## 📌 Project Overview
This project is an interactive HR Recruitment Dashboard built using Microsoft Power BI. It helps analyze recruitment performance and provides insights into candidate hiring trends.

## 📷 Dashboard Preview

![Dashboard](Screenshot%202026-07-04%20195852.png)

## 🚀 Features
- Total Candidates
- Candidate Source Analysis
- Resume Received Status
- Screening Status
- Job Position Applied
- Interview Outcome
- Final Candidate Outcome

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Visualization
- HR Analytics

## 📈 Key Insights
- 38 total candidates analyzed.
- LinkedIn was the primary source of applicants.
- Most resumes were received successfully.
- HR Intern received the highest number of applications.
- Dashboard provides a complete recruitment pipeline overview.
